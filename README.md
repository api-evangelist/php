# PHP (php)

PHP is a widely-used server-side scripting language especially suited for web development and embedded in HTML. It powers a large portion of the web including platforms like WordPress and Drupal, and provides extensive database support, framework options, and a mature ecosystem. PHP itself is a language and runtime, not a hosted API service.

## APIs

PHP is a programming language and runtime, not a hosted HTTP API. There is no OpenAPI specification because there is no REST/HTTP API to describe. Developers interact with PHP through its language manual and built-in function reference.
